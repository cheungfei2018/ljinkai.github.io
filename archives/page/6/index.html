Polymer 前端开发框架

May 29 2015
扫描二维码

初识

Polymer 是在 Google I/O 2013 发布的一个新的 Web UI 框架。2015-5-29日，也就是昨天在Google I/O 2015将Polymer升级到了1.0版本。Polymer的核心原则是 “Everything is an element”。它是一款实用、基于事件驱动、封装性和互操作性强的 Web UI 开发框架。

在了解 Polymer 之前，如果你没有了解过 Web Components ，推荐去了解下 Web Components 规范。可以通过 Web Components Wiki了解，Polymer 用到的有 Shadom Dom,Custom Elements,HTML Imports 等 Web Components 技术。
